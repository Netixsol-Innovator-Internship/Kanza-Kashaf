Introduction to HTML

  • HTML stands for HyperText Markup Language.
  • It is used to create the structure of web pages.
  • HTML elements are represented by tags.
  • Tags are enclosed in angle brackets.
  • HTML documents can contain text, images, links, and multimedia.

Key Features of HTML

  1. Semantic structure
  2. Hyperlinks
  3. Multimedia support
  4. Forms and input elements
  5. Accessibility features

HTML Tags

Tags are the building blocks of HTML. They define the structure and content of the document.

Common HTML Tags
  • <html> - The root element of an HTML page.
  • <head> - Contains meta-information about the document.
  • <body> - Contains the content of the document.
  • <title> - Sets the title of the document, displayed in the browser tab.
  • <p> - Defines a paragraph.
Conclusion

Understanding the basic structure of HTML is essential for web development. This document serves as a starting point for learning HTML.
